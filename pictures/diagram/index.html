    Diagram #

    Diagram shortcode can be used as a way of displaying the images with caption.

    {{< diagram "[path]" "css-class" >}}
**Markdown content**  
Caption
{{< /diagram >}}

    Example #

**Diagram with 50%**
{{< /diagram >}}
